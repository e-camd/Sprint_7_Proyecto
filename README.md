# Sprint_7_Proyecto

Esta aplicación extrae la información sobre los anuncios de ventas de coches
escritos en el archivo 'vehicles_us.csv'.

Genera un histograma en función del odómetro y un gráfico de dispersión del precio contra el odómetro.

## Tecnologías
- Python
- Pandas
- Plotly Express
- Streamlit

## Estructura del proyecto
- Sprint_7_Proyecto
- - ├── README.md  
- - ├── app.py  
- - ├── vehicles_us.csv  
- - ├── requirements.txt  
- - └── notebooks/  
- - - └── EDA.ipynb  

## Instalación y uso
1. Clona el repositorio
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la aplicación: `streamlit run app.py`
